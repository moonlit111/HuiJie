# 慧杰 (HuiJie) NPC 日程文档

## 基本信息

| 属性 | 值 |
|------|-----|
| 住所 | Saloon（格斯酒馆）二楼，坐标 (35, 8)，朝左 |
| 起床时间 | 6:10 (610) |
| 就寝时间 | 22:00 (2200) 或 24:00 (2400，周五/夏季周五) |
| 生日 | 秋季 10 日 |

## 日程优先级

游戏按以下顺序匹配日程，**先匹配到的生效**：

1. **特定日期**: `season_day` (如 `spring_13` — 蛋蛋节)
2. **季节+星期**: `season_dayOfWeek` (如 `summer_Mon`)
3. **星期**: `dayOfWeek` (如 `Mon`)
4. **季节+雨天**: `season_rain` (如 `summer_rain`)
5. **雨天**: `rain`
6. **季节默认**: `season` (如 `spring`)
7. **全局默认**: `spring`（本 mod 中季节默认即全局默认）

## 十、注意事项

1. **坐标需要游戏内验证**: 所有坐标 (X, Y) 是根据地图估计的，可能需要在游戏内微调以确保 NPC 站在合理位置。
2. **节日优先级最高**: 节日当天，慧杰只会出现在节日地点，不会执行常规日程。
3. **雨天覆盖季节**: 雨天日程优先于季节默认和星期日程（但低于节日）。
4. **SVE 兼容**: 部分节日坐标在安装 SVE 时会自动切换到兼容位置。
5. **婚后日程**: 结婚后慧杰会搬入农舍，日程系统会改变（由原版游戏管理，但 marriageDialogue.json 提供对话内容）。
