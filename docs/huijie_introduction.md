## 人物传记

慧杰（张慧杰），临江浅川人，毕业于西岭农业大学，研究方向为中华鲟的洄游路线追踪。毕业后进入浅川市第一中学（浅川一中）——临江省顶尖中学——担任生物教师和生物竞赛教练。

在浅川一中任教期间，慧杰带领过多届竞赛班，精心挑选种子选手重点培养，多名学生进入省队并获奖。那是他人生最骄傲的时光。

然而一切在一次研究报告中崩塌——他在关于中华鲟神经系统追踪的论文里，将海豚分类为河豚的近亲。不是笔误，他是真的分不清。更糟糕的是，他在论文中声称"把探测器套在了中华鲟的脖子上"——而鱼根本没有脖子。报告被退回，他被开除，浅川一中不要他了，研究生学历和教练生涯同时化为乌有。

最致命的打击来自2023届竞赛班。他精心挑选的四名种子选手，在省赛中全军覆没，没有一人进入全国决赛。这件事成为他内心最深的伤疤。

走投无路的慧杰来到鹈鹕镇，在格斯的酒馆二楼租了房间，开始在镇上卖桃子为生。他用过熟的桃子酿酒卖给格斯（格斯至今不知道），用卖桃子的钱维持着"研究经费"。他每天坚持做大量俯卧撑，把一切行为都冠以"操练"的名义，用表面的粗鲁和自大来掩饰内心的自卑和孤独。每次提到竞赛班，他都说"这是最后一次了"，但每次都做不到。

**核心理念**："桃子不会因为你分不清它和苹果就开除你。"

## 性格分析

### 表面人格

慧杰给人的第一印象是**粗鲁、自大、不通人情**。他见人就推销桃子，对别人的感受漠不关心，说话口无遮拦。他把一切日常行为都称为"操练"，仿佛人生就是一场永远不能停歇的训练计划。

### 内在人格

在粗鲁的外壳下，慧杰是一个**极度缺乏安全感**的人。他害怕再次失败，害怕被人发现自己的无能，所以选择先发制人——用傲慢来保护脆弱的自尊。

随着好感度提升，他逐渐展现出真实的一面：
- **2心**：有人帮捡桃子时会不知所措
- **4心**：深夜酒后坦白过去的失败
- **6心**：开始思考"不同的路不一定更差"
- **8心**：怀念当教练的时光，发现教导不一定要在教室里
- **10心**：承认"粗鲁是因为我在乎太多了"

### 关键性格特质

| 特质 | 描述 |
|------|------|
| **强迫性操练** | 每天做数百个俯卧撑，把一切行为合理化为"操练" |
| **桃子执念** | 过度热爱桃子，试图将桃子融入一切 |
| **自我防御** | 用粗鲁和自大掩饰自卑和恐惧 |
| **学术执念** | 虽然被开除，依然对鱼类研究充满热情 |
| **海豚河豚情结** | 反复提及自己分不清海豚和河豚，既是自嘲也是心结 |
| **隐藏的温柔** | 在信任的人面前展现出真诚和脆弱 |
| **创造力** | 发明桃子味鱼饵、桃子酒、桃子防虫剂等稀奇古怪的东西 |

## 人物关系

### 格斯 (Gus)
慧杰的房东和酒友。格斯收留他住在酒馆二楼，从不问为什么。慧杰把过熟桃子酿的酒卖给格斯，格斯似乎还没发现。

### 威利 (Willy)
渔业同好。威利对慧杰声称研究过鱼类持怀疑态度，尤其是听到海豚河豚的故事后。

### 文森特 (Vincent) / 贾斯 (Jas)
慧杰想把他们抓去"生物竞赛班"做操练，还曾把不太新鲜的桃子卖给文森特，被乔迪和肯特教训了一顿。

### 罗宾 (Robin)
慧杰给她送过一箱桃子，她说那是她吃过最好的桃子。慧杰知道她只是在客气。

### 艾米丽 (Emily)
慧杰曾试图给她讲解鱼类神经系统理论，她敷衍说很有趣。

### 玩家 (@)
从买卖关系的"桃子顾客"，到帮忙捡桃子的朋友，到深夜听他讲述失败故事的知己，再到最终的伴侣。慧杰在玩家面前逐渐卸下伪装，承认自己的脆弱。

## 心脏事件

### 2心事件 — 桃子摊意外

| 项目 | 详情 |
|------|------|
| **触发条件** | 好感度 ≥ 2心，晴天 |
| **地点** | Town（小镇） |
| **时间** | 全天 |

慧杰在镇上卖桃子时不小心把桃子全撒了。他嘴硬说这是"重力对桃子坠落速度影响"的实验，但还是承认自己手滑了。
- **帮他捡**：好感 +50。他会别扭地说"我自己能行"，但真心感谢你。
- **拒绝帮忙**：他说"独立完成也是一种操练"，然后自己捡。

### 4心事件 — 深夜坦白

| 项目 | 详情 |
|------|------|
| **触发条件** | 好感度 ≥ 4心 |
| **地点** | Saloon（酒吧） |
| **时间** | 21:00 - 2:00 |

深夜在酒吧，喝了自制桃子酒的慧杰讲述他的过去：在浅川一中当生物竞赛教练，毕业于西岭农业大学，研究方向是中华鲟。 核心事件——在研究报告中把海豚分类为河豚的近亲，还写了"把探测器套在中华鲟的脖子上"（鱼没有脖子）。 浅川一中不要他了。
- **选择听他讲**：好感 +75。他感激你愿意倾听，说"我觉得你不会嘲笑我"。
- **拒绝听**：好感 -10。他说"谁想听一个失败者的故事"，然后请你喝一杯。

> 此事件触发后，镇上其他NPC会开始讨论慧杰。

### 6心事件 — 山上鱼类研究

| 项目 | 详情 |

| **时间** | 全天 |

慧杰在山区假装观察鱼类（其实什么也没看到），开始质疑自己过去七年的研究生涯是否白费。
- **"每个人都有不同的路"**：好感 +75。他开始接受人生可以有不同方向。
- **"至少你还有桃子"**：好感 +50。他也认同这是实话。

### 8心事件 — 科学操练

## 日程概览

慧杰居住在格斯酒馆二楼，每天 6:10 起床。他的日程非常丰富，涵盖四季、每周七天、雨天和节日。

### 典型的一天

```
6:10   起床（Saloon 二楼）
7:00   晨练俯卧撑（Saloon 内）
8:30   外出活动（镇上/山区/森林/海滩，依日程而定）
12:00  午间活动（卖桃子/看鱼/研究）
15:00  下午活动（地点依日程而变化）
18:00  回酒吧喝酒（Saloon）
22:00  就寝（Saloon 二楼）
```

### 各季节特点

- **春季**：镇上卖桃子 + 山区看鱼，中规中矩
- **夏季**：频繁去海滩研究海洋鱼类，下午基本泡在海边
- **秋季**：丰收季最忙，镇上和山区来回奔波
- **冬季**：大部分时间待在酒吧看书和操练，只下午出门散步
- **雨天**：全天不出门，在酒吧发呆 → 俯卧撑 → 研究桃子酒配方

### 特殊日程

- **周一**：去皮埃尔杂货店（试图摆桃子摊）
- **周二**：去森林观察玛妮的牛 + 海滩
- **周四**：山区日，整天看鱼做俯卧撑
- **周五**：唯一熬夜到 24:00 的日子（在酒吧喝酒）
- **周六**：森林日，观察溪流和淡水鱼
- **周日**：放松日，看书 + 和格斯聊桃子酒销路

### 节日出席

慧杰会出席全部 8 个鹈鹕镇节日：蛋蛋节、花舞节、夏威夷宴会、月光水母之舞、星露谷展览会、万灵节、冰雪节、冬日星盛宴。

## 结婚后

婚后慧杰会搬入农舍，但他依然保持着操练习惯和桃子执念：

### 婚后日常

- **早晨**："我刚做完五百个俯卧撑。你要不要也来几个？我可以从教练模式切换到配偶模式。"
- **晚上**："辛苦了一天吧？来一杯特制桃子茶？不是卖的那种版本。"
- **户外**："这个院子很适合做户外操练。你看那边那棵树，完美的引体向上训练点。"
- **雨天**："以前一个人在酒馆的雨天特别孤独。现在...好像也没那么糟了。"

### 婚后性格变化

婚后的慧杰明显变得柔软，但依然保持着自我：
- 承认"配偶时间比操练计划更重要"
- 自嘲当年的论文错误时更多是释然而非痛苦
- 对未来有了新的规划（种桃树、写书）
- 偶尔还是会试图把过期桃子做成"特制食品"

## 趣闻轶事

- 慧杰把一切行为都称为"操练"——捡桃子是手眼协调操练，喝酒是味蕾操练，看书是大脑操练
- 他声称发明了"桃子味鱼饵"，效果很好但钓上来的鱼都有桃子味
- 他曾用矿泉水和河水分别浇灌桃树做对比实验
- 他给桃树唱歌，虽然不确定这是否科学
- 他在研究既有桃子甜度又有鱼类 omega-3 脂肪酸的超级水果
- 他曾把桃子冰淇淋配方搞炸了冰箱
- 他分不清海豚和河豚这件事在镇上已经人尽皆知
- 他从不喝自己酿的桃子酒，但很乐意卖给别人
- 他把过期桃子做成"特制操练补充饮料"（其实就是酒）
- 他的终极梦想是组建一支生物竞赛国家级队伍
- 他来自临江浅川，经常说临江方言"那能中"（那怎么可以）但星露谷没人听得懂
- 他曾经把探测器套在中华鲟的脖子上（鱼没有脖子）并把这段经历写进了论文
- 他的学生问他"老师，鱼有脖子吗"的时候，整个实验室安静了三十秒
- 其他NPC说他的样子像加拉帕戈斯象龟，脖子很长
- 他总是说"这是最后一次带竞赛班了"但每次都做不到
- 2023届竞赛班种子选手全军覆没是他最深的创伤
