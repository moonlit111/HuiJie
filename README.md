### 人物传记

慧杰曾是一名前途光明的水产研究生，毕业后在全国中学生生物学联赛担任教练，带领十二名学生参加比赛，其中三人获得省一等奖。那是他人生最骄傲的时光。

然而一切在一次研究报告中崩塌——他在关于海洋生物神经系统的论文里，将海豚分类为河豚的近亲。不是笔误，他是真的分不清。报告被退回，他被开除，研究生学历和教练生涯同时化为乌有。

走投无路的慧杰来到鹈鹕镇，在格斯的酒馆二楼租了房间，开始在镇上卖桃子为生。他用过熟的桃子酿酒卖给格斯（格斯至今不知道），用卖桃子的钱维持着"研究经费"。他每天坚持做大量俯卧撑，把一切行为都冠以"操练"的名义，用表面的粗鲁和自大来掩饰内心的自卑和孤独。

**核心理念**："桃子不会因为你分不清它和苹果就开除你。"

### 性格分析

**表面人格**：粗鲁、自大、不通人情。见人就推销桃子，把一切行为都称为"操练"。

**内在人格**：极度缺乏安全感，害怕再次失败，用傲慢保护脆弱的自尊。随着好感度提升逐渐展现真实的一面：

- **2心**：有人帮捡桃子时会不知所措
- **4心**：深夜酒后坦白过去的失败
- **6心**：开始思考"不同的路不一定更差"
- **8心**：怀念当教练的时光，发现教导不一定要在教室里
- **10心**：承认"粗鲁是因为我在乎太多了"

### 日程概要

慧杰住在格斯酒馆二楼，每天 6:10 起床：

- **周一**：皮埃尔店 → 镇上卖桃子 → 山区 → 回酒吧
- **周二**：森林做俯卧撑 → 镇上 → 海滩研究鱼 → 回酒吧
- **周三**：酒吧看书 → 镇上 → 山区操练 → 回酒吧
- **周四**：山区看鱼 → 山区操练 → 镇上卖桃子 → 回酒吧
- **周五**：镇上卖桃子 → 海滩 → 酒馆喝酒（晚归到 24:00）
- **周六**：森林观察 → 森林操练 → 回酒吧
- **周日**：酒吧看书 → 镇上 → 酒吧和格斯聊天
- **雨天**：全天待在酒馆不出门
- **冬季**：多在室内，起床较晚

## 模组功能一览

- 全新可结婚 NPC（自定义精灵图 + 肖像）
- ~300 行中文对话（按季节/星期/心数/天气区分）
- 5 个心脏事件（2/4/6/8/10 心，含分支选择）
- 丰富日程系统（7 天/4 季/雨天/节日全覆盖）
- 完整结婚系统（订婚/婚礼/婚后对话 ~70 条）
- 9 个自定义动画（俯卧撑/读书/卖东西/喝酒等）
- 6 个特殊任务（Special Orders）
- 10 个其他 NPC 互动对话
- 信件系统（欢迎信/桃子礼物/配方赠送）
- 8 个节日出席（含 SVE 兼容）

## 安装方法

1. 安装 [SMAPI](https://smapi.io/) 和 [Content Patcher](https://www.nexusmods.com/stardewvalley/mods/1915)
2. 下载本模组，将 `HuiJie` 文件夹放入 `Mods` 目录
3. 启动游戏，**睡一天** 后慧杰会出现在格斯酒馆并开始按日程移动
